# 🐾 하리 갤러리 (Hari's Gallery)

반려견 **하리**의 성장일기를 담은 웹페이지 — HTML + CSS + JS 미니프로젝트

## 1. 프로젝트 개요

- 프로젝트 이름 : 하리 갤러리
- 주제 : 하리가 어렸을 때부터 지금까지의 성장일기
- 기획 의도 : 하리가 성장하면서 기억에 남는 대표 사진들을 한곳에 모아보기

## 2. 핵심 기능 3가지

1. 좋아요 모으기
   - 상단 header의 `좋아요 ❤️` 버튼을 클릭하면 좋아요 갯수가 누적됨
   -> localStorage를 사용하여 새로고침시 초기화 안되게
2. 사진 추가하기
   - 하단 카드의 화살표(↗)를 눌러 사진 파일을 불러옴
   - "새로운 추억이 생겼나요?" 영역이 불러온 사진 이미지로 바뀜
   ->(조건문 사용 이미 올라가있는 파일이면, 다른사진 선택 문구)
   - `사진 추가하기`를 누르면 레코드 부분에 사진이 추가됨
   -> (레코드 안 클래스에 객체로 들어가게끔)
   - 올린 사진은 축소(canvas)해서 localStorage에 저장 → 새로고침해도 유지됨
3. 레코드 사진 넘겨보기
   - 레코드 카드에서 사진들을 화살표로 넘겨가면서 볼 수 있음 (3D 원형 캐러셀)
   (레코드 사진 배열을 클래스로 지정, 정면 사진 순번도 localStorage에 저장)

## 2-1. 추가 구현

- 휴지통(🗑️) 버튼 : 올린 사진이 정면에 왔을 때만 활성화되어 캐러셀에서 삭제
  (기본 사진 5장은 지워지지 않게 보호, 삭제도 localStorage에 반영)
- 사진 개수가 늘어나면 캐러셀 반지름이 자동으로 커져서 사진끼리 안 겹침

## 3. 화면 구성


- header : 로고, 좋아요 버튼(핵심기능 1), 네비게이션 링크
- 레코드 카드 : 캐치프레이즈 + 사진 캐러셀 (핵심 기능 2, 3)
- 대표 사진 카드 : 하리의 대표 사진
- 소개 카드 : 하리 갤러리 소개글
- 사진 추가 카드 : 새 사진 업로드 진입점 (핵심 기능 2)
- 투두리스트 카드 : 하리의 하루 일과 
- 소셜 카드 : 인스타그램 · 유튜브 링크
